Algunos lenguajes de la web

HTML + CSS + Javascript

  • HTML5

    HTML 5(HyperText Markup Language, version 5) es la quinta versión del lenguaje básico de la World Wide Web, HTML.

    Usa la extensión .html

  • CSS

    Hoja de estilo en cascada o CSS (siglas en ingles de cascading style sheets) es un lenguaje usado para definir y crear la presentación de un documento estructorado escrito en HTML

    Usa la extención css

  • JS

    Javascript(abreviado comunmente "JS" es un lenguaje de programación interpretado principalmente en el lado del cliente (client-side), implementado como parte de un navegador web permitiendo mejoras en la interfaz de usuario y páginas webs dinámicas

    Usa la extención .js
